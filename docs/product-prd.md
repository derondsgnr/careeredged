CareerEdge™ Comprehensive Product Requirements Document (PRD)

 

1. Executive Summary
CareerEdge™ is a multi-sided career development ecosystem designed to democratize professional advancement. Unlike traditional job boards, CareerEdge™ uses an Archetype-Based Matching Model and Rule-Based Conversational AI (Sophia) to guide users through personalized career roadmaps. It serves six distinct user types, creating a networked environment where seekers, supporters, educators, and funders interact seamlessly.

 

 

2. The 6 User Roles & Deep-Dive Journeys
 

2.1 EdgeStar (Individual Seeker)
*   Journey: Onboarding (Resume/Sophia) → Archetype Assessment → Goal Setting (SMART) → Roadmap Generation → Execution (EdgeProd) → Mentorship/Job Placement.

*   Critical Feature: Career Archetypes. Uses a 15-question model to map users to 18 archetypes (e.g., *Innovator, Visionary, Helper, Strategist*).

*   Edge Case: Users with "Career Gaps" or "Career Pivots" receive specialized transition logic.

 

2.2 EdgeParent (Supporter)
*   Journey: Sign-up → Link to Child (QR) → Involvement Level Setting → Read-Only Progress Tracking.

*   Involvement Levels: *Very Active* (Daily) to *Observer* (Occasional).

*   Designer Note: UI must emphasize trust and shared achievement without compromising child autonomy.

 

2.3 Educational Institution (The Educator)
*   Journey: Org Setup → Student Connection → Event/Career Fair Creation → Analytics Dashboard.

*   Key Action: QR Check-ins. Automates attendance tracking for physical and virtual workshops.

*   Data Need: Student adoption rates of AI roadmaps to measure "Career Readiness."

 

2.4 Coach/Mentor (The Guide)
*   Journey: Profile Verification → Portfolio Building → Session Scheduling → Earnings Tracking.

*   Monetization: Supports both *Volunteer Mentors* (Edge Mentor) and *Paid Professionals* (Edge Coach).

 

2.5 NGO & Government Agency (The Funders/Partners)
*   Journey: Official Org Setup → Program/Grant Creation → Participant/Recipient Tracking → Impact Reporting.

*   System Logic: Grants can be applied for by NGOs/Institutions; Programs are enrolled in by EdgeStars.

 

 

3. Core Feature Deep-Dive
 

3.1 Sophia: The Onboarding Engine
*   Logic: A rule-based state machine that handles 6 unique branching flows.

*   Cost Strategy: $0 AI cost during assessment by using predefined options (800+ professions, 400+ skills).

*   UI Pattern: Fixed bottom-right chat bubble with a 400px wide panel, animated avatar, and progress tracking.

 

3.2 EdgePath: Career Discovery
*   The Assessment: 15 weighted questions covering Legacy, Work Style, Problem Solving, Values, and Environment.

*   Matching Algorithm: Normalizes archetype scores to calculate a "Match Percentage" (capped at 99%) against a career database.

*   Output: Best-fit career cards with salary, demand, and "Time to Transition" metadata.

 

3.3 EdgeProd: The Productivity Suite
*   Goal Wizard: A 5-step SMART Goal builder (Specific, Measurable, Achievable, Relevant, Time-bound).

*   Execution Tools:

*   Pomodoro: Focused work intervals with stats.

*   OKRs: High-level objective tracking.

*   Accountability: QR-based session pairing with Edge Buddies.

 

3.4 ImmigrationEdge (Specialized Workflow)
*   Purpose: Guides international seekers through visa/career pathways.

*   Logic: Questionnaire-driven roadmap that calculates "Pathway Costs" and "Support Recommendations."

 1. Product Overview
CareerEdge is a comprehensive career development ecosystem that connects individual career seekers (EdgeStars) with parents, mentors, educational institutions, NGOs, and government agencies. It leverages a rule-based AI chatbot (Sophia) for cost-efficient onboarding and a QR-based connection system for instant networking.

3. Feature Inventory & Workflows
 

3.1 Onboarding & Assessment (Sophia Chatbot)
Goal: Gather user data with $0 AI cost using a rule-based state machine.

• **Workflow (EdgeStar):**
• **Option A:** Upload Resume → AI Role Extraction → Manual Role Verification → Goal Selection.
• **Option B:** Chat with Sophia (11 questions: name, profession, experience, skills, education, industry, satisfaction, motivations, goal, learning style, time commitment).
• **Workflow (Others):** Specialized 3-6 question flows for Parents (relationship, involvement), Institutions (type, size), Coaches (specialization, rate, capacity), etc.
 

3.2 Instant Connections (QR System)
Goal: Enable zero-friction networking.

• **Workflows:**
• **Edge Buddy Match:** Generate/Scan QR to connect with a peer for accountability.
• **Parent-Child Link:** Parent scans child's QR to view read-only roadmap.
• **Event Check-in:** Student scans event QR (or Institution scans student QR) for attendance.
• **Profile Share:** Scan to view public profile/portfolio.
 

3.3 Career Development Tools
• **Roadmap Generator:** (AI-Powered) Creates a personalized career path based on assessment.
• **ResumeEdge:** Resume builder, bullet enhancer, and content review.
• **EdgeProd:** Productivity hub with Pomodoro timer, goals (OKR), and accountability sessions.
• **ImmigrationEdge:** Specialized pathway for international career seekers (questionnaire → roadmap).
 

3.4 Marketplaces & Directories
• **Coach Marketplace:** Browse/Filter/Search coaches, view portfolios, and book sessions.
• **NGO Program Directory:** Browse active programs, filter by audience, and enroll.
• **Government Grant Portal:** Browse grants, view deadlines, and apply (NGO/Institutions).
• **Job Board:** Filter by career path and location, apply directly.
